# python-asyncio-test

## Getting Started

### Prerequisites
- Python 3.6 over 
	- requests
	- aiohttp
	
### Installing

```bash
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
```
